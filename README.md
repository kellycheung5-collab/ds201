# GitHub Sandbox

Welcome to your Git testing ground!

Use this repository to practice cloning, branching, and pushing code.

1. Clone this repository to your local machine.
2. Create a new branch: `git checkout -b feature/add-profile`

### Task 1: Create your profile folder
1. Inside the `students` directory, create a new folder with your name.
2. Inside your new folder, create files.

### Task 2: Propose your changes
1. Commit and push your branch to your forked repository.
2. Open a Pull Request back to the main repository.
